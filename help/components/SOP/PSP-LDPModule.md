# PSP / LDP Module
PSP Admin - takes care of all actions in which members pertaining to PSP level are concerned. 
While, LDP Admin - takes care of actions in which LDP members are concerned. 

Major responsibilities and features - 
1. Review and Publish an SOP
2. Review and Publish a Role Profile
3. Create and Schedule Trainings
4. Publish Trainings and Evaluations
5. Publish RP Assessments
6. Publish results and ratings
7. Schedule for RP Matrix ratings
8. Manage base profiles for each member
9. Set type of rating for each member
10. Set Permissions / Trainers role for members
11. Send New Notices / News for members from Communication module.
12. Add new user - when a new user is added to Pakka 
13. Modify / refresh user permissions / access when any user's base profile is changed.
